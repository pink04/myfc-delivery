폼 액션

common.css — 기본(주 버튼 전폭) · form-actions-sub(주 버튼 아래 보조 행 균등 분할) · form-actions-split(2열: 보조+주, 640px 이하 세로 스택). 모달 하단은 .c-modal-btns가 유사 그리드입니다.

폼 맨 아래 CTA는 .form-actions로 감싸고, 주 버튼(.btn.btn-primary)은 직계 자식으로 둡니다. 상단 여백·주 버튼 전폭은 common.css에서 처리합니다. 네이밍 요약은 guide/naming-conventions.md 2)절을 참고하세요.

1) 기본 — 주 버튼 한 줄(전폭)

가입·문의 등 단일 CTA. .form-actions 직계 자식 .btn.btn-primary

<div class="form-actions">
  <button type="submit" class="btn btn-primary">…</button>
</div>

2) 주 버튼 + 보조 행 — form-actions-sub

확인 등 전폭 주 버튼 아래, 보조 버튼을 한 행에 균등 배치 (상담 상세 MFC005_L01_03_01 등).

<div class="form-actions">
  <button type="button" class="btn btn-primary">확인</button>
  <div class="form-actions-sub">
    <button type="button" class="btn btn-sub">…</button>
    <button type="button" class="btn btn-sub">…</button>
  </div>
</div>

3) 2열 스플릿 — form-actions form-actions-split

한 가지 스타일입니다. 좁은 보조(보통 a.btn 링크) + 넓은 주 버튼(.btn-primary). 화면마다 라벨만 다릅니다. 예: 마이페이지 MFC005_L01_01 — 「내 프로필…」+「수정 완료」 / 단계 폼 — 「이전」+「다음」. 오른쪽은 제출이면 type="submit"으로 둘 수 있습니다.

이전 
<div class="form-actions form-actions-split">
  <a href="…" class="btn">보조(링크/이전 등)</a>
  <button type="submit" class="btn btn-primary">주요 액션</button>
</div>
